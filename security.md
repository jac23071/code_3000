# Intended Users of This Repo:

The only people intended to have access to and use this repo are myself and the Professor of CSE 3000 and
any teaching authority he employs in the classroom (i.e. a TA)

# Security Risks:

Should access to this repo fall into the wrong hands, they may alter/destroy my work as an act of sabotage
and/or vandalism.

# Security Measures

To prevent all possibility of tampering, I have my CSE 3000 repo set so that none but I have access to edit
this repo. As a redundant layer of protection, I have employed a ruleset for pull requests should I give another person access to edit the repo in the future.
